# Quiz: Fine-tuning Techniques

## Questions

### Question 1
What is the rank (r) parameter in LoRA?

A) The number of training epochs
B) The rank of the adaptation matrices
C) The learning rate
D) The batch size

### Question 2
Which technique uses 4-bit quantization?

A) LoRA
B) QLoRA
C) Full Fine-tuning
D) Prefix Tuning

### Question 3
What is catastrophic forgetting?

A) When the model becomes too large
B) When fine-tuning causes loss of original capabilities
C) When training data is lost
D) When the model forgets to generate output

### Question 4
Which PEFT method adds trainable tokens to input?

A) LoRA
B) Adapter
C) Prefix Tuning
D) Quantization

### Question 5
Approximately how much GPU memory does QLoRA need for a 7B model?

A) 14 GB
B) 7 GB
C) 3 GB
D) 1 GB

### Question 6
What is the advantage of LoRA adapters?

A) They cannot be swapped
B) They are modular and can be changed
C) They require more memory
D) They are slower

### Question 7
Which technique would you use to preserve base model capabilities?

A) Full Fine-tuning
B) LoRA with regularization
C) QLoRA
D) Prefix Tuning

### Question 8
What does the lora_alpha parameter control in LoRA?

A) Dropout rate
B) Learning rate scaling
C) Model size
D) Batch size

## Answers

1. B) The rank of the adaptation matrices
2. B) QLoRA
3. B) When fine-tuning causes loss of original capabilities
4. C) Prefix Tuning
5. C) 3 GB
6. B) They are modular and can be changed
7. B) LoRA with regularization
8. B) Learning rate scaling
