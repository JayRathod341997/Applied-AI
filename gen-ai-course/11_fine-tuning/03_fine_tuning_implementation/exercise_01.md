# Exercise: Setting Up Fine-tuning Pipeline

## Objective

Create a complete fine-tuning pipeline using Hugging Face.

## Requirements

Create a Python script that:
1. Prepares training data
2. Configures the model
3. Sets up training arguments
4. Runs training
5. Saves the model

## Sample Data Format

```json
{"instruction": "Summarize this", "input": "Long text...", "output": "Short summary..."}
```

## Time Duration

45 minutes
