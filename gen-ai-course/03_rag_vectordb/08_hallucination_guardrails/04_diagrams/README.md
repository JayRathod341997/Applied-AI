## Defense in Depth Model

```
┌─────────────────────────────────────────────────────────────────┐
│              DEFENSE IN DEPTH MODEL                            │
└─────────────────────────────────────────────────────────────────┘

Layer 1: Retrieval Quality
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  Query ──▶ [Hybrid Search] ──▶ [Reranker] ──▶ [Quality Check]  │
│                                                              │
│  ✓ Increase top-k                                             │
│  ✓ Use hybrid (dense + sparse) search                         │
│  ✓ Add reranking                                               │
│  ✓ Filter by metadata                                        │
│                                                              │
└──────────────────────────────────────────────────────────────┘
                              │
                              ▼
Layer 2: Prompt Engineering  
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  ┌─────────────────────────────────────────────────────┐       │
│  │ SYSTEM PROMPT                                       │       │
│  │ 1. Only use information from provided context     │       │
│  │ 2. If unsure, say so                             │       │
│  │ 3. Cite sources                                  │       │
│  │ 4. Don't add information not in context        │       │
│  └─────────────────────────────────────────────────────┘       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
                              │
                              ▼
Layer 3: Generation Control
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  - Use temperature = 0 for factual queries                   │
│  - Add max_tokens limits                                     │
│  - Use factual-focused models                               │
│                                                              │
└──────────────────────────────────────────────────────────────┘
                              │
                              ▼
Layer 4: Guardrail Evaluation
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│                    ┌──────────────────────┐                │
│                    │  EVALUATE OUTPUT      │                │
│                    └──────────┬───────────┘                │
│                               │                             │
│              ┌────────────────┼────────────────┐            │
│              ▼                ▼                ▼            │
│     ┌──────────────┐  ┌──────────────┐  ┌──────────────┐   │
│     │ Groundedness │  │ Relevance    │  │ Consistency  │   │
│     │    ≥ 0.7    │  │    ≥ 0.6     │  │    ≥ 0.8    │   │
│     └──────┬───────┘  └──────┬───────┘  └──────┬───────┘   │
│            │                 │                 │            │
│            ▼                 ▼                 ▼            │
│     ┌─────────────────────────────────────────────────────┐  │
│     │            All checks pass?                        │  │
│     │         YES ──▶ Return to User                     │  │
│     │         NO  ──▶ Fallback Chain                     │  │
│     └─────────────────────────────────────────────────────┘  │
│                                                              │
└──────────────────────────────────────────────────────────────┘
                              │
                              ▼
Layer 5: Monitoring
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  - Log all queries and evaluations                          │
│  - Track metrics over time                                 │
│  - Alert on degradation                                    │
│  - Continuously improve                                   │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```
