# LangChain Overview

This sub-module covers the introduction and overview of LangChain framework.

## Topics Covered

- Introduction to LangChain
- LangChain vs other frameworks
- Core components and architecture
- Installation and setup

## Learning Objectives

By the end of this topic, you will:
- Understand what LangChain is and why it's used
- Know the core components of LangChain
- Be able to set up a basic LangChain environment
- Understand the difference between LangChain and LangGraph

## Prerequisites

- Python 3.8+
- Basic understanding of LLMs
- API keys for LLM providers (OpenAI, Anthropic, etc.)
