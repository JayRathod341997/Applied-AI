# Exercise: Building Blocks

## Objective

Create a chain using prompt templates and output parsers.

## Instructions

1. Create a prompt template with at least 2 variables
2. Use an output parser (JSON or Pydantic)
3. Chain them together with an LLM
4. Test with different inputs

## Expected Output

A Python script demonstrating:
- Prompt template creation
- Output parsing
- Chain invocation with parsed output
