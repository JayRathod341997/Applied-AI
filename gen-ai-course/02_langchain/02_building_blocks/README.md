# Building Blocks

This sub-module covers the fundamental building blocks of LangChain.

## Topics Covered

- Chat Models and LLMs
- Prompt Templates
- Output Parsers
- Chains fundamentals

## Learning Objectives

By the end of this topic, you will:
- Understand different chat models and LLM wrappers
- Master prompt template creation
- Learn output parsing techniques
- Build basic chains

## Prerequisites

- Completion of LangChain Overview
- Understanding of Python
