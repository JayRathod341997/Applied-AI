# Exercise: Chat Models and Chains

## Objective

Create a sequential chain that:
1. Takes a topic as input
2. Generates 3 facts about the topic
3. Summarizes the facts into a brief paragraph

## Instructions

1. Create two LLMChains: one for facts, one for summary
2. Combine them using SequentialChain
3. Test with different topics

## Expected Output

A working sequential chain that produces summarized output
