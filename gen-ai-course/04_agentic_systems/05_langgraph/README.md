# LangGraph Framework

## Overview

This module covers LangGraph, a powerful framework for building stateful, multi-agent applications with Large Language Models. LangGraph extends LangChain with graph-based workflows, enabling complex agentic behaviors with cycles, conditional routing, and human-in-the-loop interactions.

## Learning Objectives

By the end of this module, you will:
- Understand the core concepts of LangGraph (Nodes, Edges, State)
- Build stateful conversational agents
- Implement agentic design patterns (Reflection, Tools, Planning-ReAct)
- Create multi-agent collaboration workflows
- Handle human-in-the-loop (HITL) interactions
- Debug and visualize LangGraph applications

## Topics Covered

### Introduction to LangGraph
- What is LangGraph and why use it?
- Key concepts: Nodes, Edges, State
- LangChain vs LangGraph comparison
- Full walkthrough of building a LangGraph-based agent

### Implementing Agentic Design Patterns
- Reflection patterns
- Tool integration
- Planning-ReAct patterns
- Multi-Agent collaboration

### Working with Memory and State
- Storing and retrieving conversation state
- Using Vector Stores for persistent memory
- Balancing short-term vs. long-term memory
- Implementing contextual persistence between nodes

### Integrating Tools and Human-in-the-Loop
- Registering and using external tools
- Designing interruptible workflows
- Implementing HITL checkpoints
- Building Reflective Research Agent with HITL Interrupt
