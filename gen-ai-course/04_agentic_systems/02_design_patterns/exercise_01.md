# Exercise: Implementing Agent Design Patterns

## Objective

Implement different agent design patterns including reactive vs planning agents, reflection, and memory patterns.

## Requirements

1. **Part A**: Implement a reactive agent for order status checking
2. **Part B**: Implement a planning agent for travel booking
3. **Part C**: Add reflection to improve agent responses
4. **Part D**: Implement multiple memory patterns

## Scenario

Build an e-commerce agent system with:
- A reactive component for simple queries
- A planning component for complex workflows
- Reflection for quality assurance
- Different memory types for context retention

## Time Duration

45 minutes

## Expected Output

A Python script demonstrating:
1. Reactive agent pattern
2. Planning agent pattern
3. Reflection mechanism
4. Buffer and vector memory implementations

## Hints

- Start with simple if/else for reactive
- Use function calling for planning
- Add self-evaluation step after generating response
- Use lists for buffer memory, dictionaries for quick lookup
