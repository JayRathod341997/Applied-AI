# Agentic AI Design Patterns

## Overview

This module covers the essential design patterns for building Agentic AI systems, including reactive and planning agents, reflection, tools, actions, and memory patterns.

## Learning Objectives

- Understand reactive vs planning agents
- Learn reflection patterns for agent improvement
- Design and integrate tools with agents
- Implement various memory patterns
- Apply ReAct (Reasoning + Acting) patterns

## Key Topics

1. **Reactive and Planning Agents**
   - Understanding the differences
   - When to use each type

2. **Reflection Patterns**
   - Self-evaluation and improvement
   - Implementation techniques

3. **Tools Integration**
   - Designing tools
   - Tool invocation patterns

4. **Memory Patterns**
   - Buffer memory
   - Sliding window
   - Summary memory
   - Vector memory
   - Scratchpad
   - Shared memory

5. **Planning-ReAct**
   - Reasoning + Acting patterns

## Prerequisites

- Basic understanding of Agentic AI
- Python programming experience
- Familiarity with LangChain or similar frameworks
