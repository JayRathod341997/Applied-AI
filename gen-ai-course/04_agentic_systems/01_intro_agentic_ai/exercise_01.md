# Exercise: Building Your First Agent

## Objective

Create a simple Agentic AI system that can process user requests and take autonomous actions.

## Requirements

1. Create a basic agent with the following components:
   - Goal definition
   - Perception module
   - Reasoning capability
   - Action execution
   - Feedback mechanism

2. The agent should be able to:
   - Accept a user request
   - Determine what information is needed
   - Take action to gather information
   - Provide a response

## Scenario

Build an agent that can help users find information about products in an inventory. The agent should:
- Ask for clarifying questions if needed
- Search through product data
- Return relevant results with details

## Time Duration

30 minutes

## Hints

- Start with a simple goal definition
- Use if/else logic for basic reasoning
- Create mock data for product inventory
- Focus on the agent loop: perceive → reason → act → feedback
