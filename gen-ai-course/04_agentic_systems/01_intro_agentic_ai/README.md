# Introduction to Agentic AI

## Overview

This module covers the fundamentals of Agentic AI systems, their components, and how they differ from traditional AI approaches.

## Learning Objectives

- Understand what Agentic AI is and its key characteristics
- Learn the foundational capabilities of Agentic AI
- Explore the differences between prompts and Agentic AI
- Understand the components that make up Agentic AI systems
- Build foundational knowledge for advanced agent patterns

## Key Topics

1. **What is Agentic AI?**
   - Autonomous decision making
   - Goal-oriented behavior
   - Adaptive learning capabilities

2. **Foundational Capabilities**
   - Autonomy
   - Reasoning
   - Action

3. **Components of Agentic AI**
   - Goals
   - Perception
   - Reasoning
   - Plan
   - Act
   - Feedback
   - Memory

## Prerequisites

- Basic understanding of LLMs
- Python programming experience
- Familiarity with API interactions
