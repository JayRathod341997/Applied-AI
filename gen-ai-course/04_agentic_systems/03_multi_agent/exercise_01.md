# Exercise: Building a Multi-Agent System

## Objective

Create a multi-agent system where agents collaborate to complete a research task.

## Requirements

1. Create 3 specialized agents:
   - Planner Agent
   - Search Agent
   - Synthesizer Agent
2. Implement communication between agents
3. Coordinate agent activities

## Scenario

Build a research team that:
- Plans research tasks
- Gathers information
- Synthesizes findings into a report

## Time Duration

30 minutes
